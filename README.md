# 🛒 Racim Shop - مشروع متجر إلكتروني

مرحباً! هذا هو مشروع متجري الإلكتروني الأول **Racim Shop**. سيت متكامل يخدم بـ **Firebase** لتسيير المستخدمين والمنتجات.

## 🚀 المميزات:
* **نظام مستخدمين:** (مشتري / بائع).
* **تفعيل يدوي:** الحسابات ما تخدمش حتى يفعلها الأدمن (رسيم).
* **قاعدة بيانات مباشرة:** مربوط بـ Realtime Database.
* **واجهة عصرية:** ديزاين مريح للعين وسريع.

## 🛠️ التكنولوجيا المستعملة:
* **HTML5 / CSS3** للواجهة.
* **JavaScript** للمنطق (Logic).
* **Firebase** (Auth & Database) للخلفية (Backend).

## 📂 ملفات المشروع:
* `index.html`: واجهة المتجر وتسجيل الدخول.
* `admin.html`: لوحة تحكم رسيم لتفعيل الحسابات.
* `README.md`: وصف المشروع.
